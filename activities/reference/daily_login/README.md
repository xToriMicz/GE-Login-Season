# อ้างอิงสถานะ Daily Login — เช็คแค่ 2 รูป

เก็บรูป **2 รูป** เป็นดาต้าเบสอ้างอิง — สองสถานะต่างกันสิ้นเชิง ไอดีเยอะ ถ้าตรวจจับผิดพังหมด

## รูปดาต้าเบส (2 รูป)

| รูป | ไฟล์ | หน้าตา | การกระทำ |
|-----|------|--------|----------|
| **รูปที่ 1** | `state_received.png` | รับแล้ว — วงกลมทอง + ✓ + ข้อความ "รับไอเทมแล้ว" (ไม่มีแถบเขียวแบบ claimable) | **ไม่คลิก** |
| **รูปที่ 2** | `state_claimable.png` | สามารถรับได้ — มี**แถบเขียว** (DAY x) | **คลิก** |

สองรูปมีความต่างกันสิ้นเชิง โค้ดเทียบจากข้อความ + แถบเขียว

## กฎตรวจจับในโค้ด

1. **รับแล้ว (รูปที่ 1)**  
   ตรวจจาก: slot มีข้อความ **"รับไอเทมแล้ว"** → **ไม่คลิก**

2. **รับวันนี้ได้ (รูปที่ 2)**  
   ตรวจจาก: **ไม่มี** "รับไอเทมแล้ว" และ slot มี **แถบเขียว** → **คลิก**

โค้ดตรวจจับ: `activities/daily_login_detection.py`
